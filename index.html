My projects

Mapa project

Mapa is an open-source interactive map for local initiatives and volunteers. It is based on React + Redux, and uses OpenStreetMaps API to fetch map data. Users can add customizable pins on a map or search for existing entries.

React Redux Eslint

Cream app

A React-based kanban board with Redux for state management. Supports both Russian ang English localisations. Columns and tasks are moved with drag-and-drop with optimistic updates. I also wrote back-end for this app using Node, Express and PostgreSQL.

React Redux Express PostgreSQL Typescript

Language Hike

A web app for learning English. Includes a set of educational language games, a schoolbook and a personal dictionary. Authorized users can track their progress and statistics. Created with React, MUI and Typescript.

React Redux MIU Typescript

Home library service

A NestJS application for managing music collection. TypeORM is used for communication with Postgres DB. Includes auth logic with refresh and access tokens. App and DB are composed with Docker.

NestJS Docker TypeORM Typescript

Landing page for new devices

A simplistic landing page for a new line of electronic measuring devices from JSC MNIPI. The design was inspired by suprematism and early soviet-era posters.

HTML SCSS BEM Figma

Node chat app

A simple online chatting app made with Node.js and Express. Users can create or join existing rooms, send text messages and share geolocation data.
